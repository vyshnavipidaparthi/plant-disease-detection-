About Plant Disease Detection

Project Overview

The Plant Disease Detection system is an advanced machine learning application that helps detect plant diseases using deep learning techniques. By analyzing images of plants, this system can classify and identify diseases in various crops, such as tomatoes, peppers, and potatoes. The goal of the project is to provide a reliable, quick, and easy way for farmers and plant enthusiasts to detect and address diseases in plants.

Key Features
  • Detects various plant diseases with high accuracy using trained AI models.
  • Supports multiple plant species, including tomato, pepper, and potato.
  • Easy-to-use interface with image upload functionality for disease prediction.
  • Supports two trained models for disease classification: Best Model and Final Model.
  • Real-time disease detection and classification with confidence scores.
Technology Stack

The Plant Disease Detection system uses cutting-edge machine learning and deep learning techniques to accurately identify plant diseases. The system is built using the following technologies:

  • Deep Learning Framework: TensorFlow & Keras
  • Model Training: Hybrid deep learning models for improved accuracy
  • Programming Language: Python
  • Web Framework: Flask for the web application
  • Front-End: HTML, CSS, JavaScript (Bootstrap for responsive design)
  • Image Processing: PIL (Pillow) for image handling and preprocessing
About Dataset

This dataset is recreated using offline augmentation from the original dataset. The original dataset can be found on this GitHub repository. The dataset consists of approximately 87K RGB images of healthy and diseased crop leaves, categorized into 38 different classes. The total dataset is divided into an 80/20 ratio for training and validation, preserving the directory structure. Additionally, a new directory containing 33 test images was created for prediction purposes.

Content
  • Train: 70,295 images
  • Test: 33 images
  • Validation: 17,572 images
How It Works

The process is simple:

  1. Upload an image of the plant.
  2. Select one of the trained models (Best Model or Final Model) for disease detection.
  3. The model processes the image and predicts the disease with a confidence score.
  4. The predicted disease and confidence level are displayed on the result page.
